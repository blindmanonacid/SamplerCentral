Global Setup


Global Setup is accessed by clicking the black 'global setup' button near the bottom of the menu where you load new samplers.



Sound Card Connection Priority


Here you can choose what soundcard Sampler Central should connect to if it is present. This is uselful if you work with different soundcards and dont want to mess around with settings at startup.



Output Bus Channels


The number of internal channels Sampler Central uses. It should be set to the same or more than your soundcard has outputs. This is not displaying channels available in the soundcard.



Input Bus Channels


The number of internal channels Sampler Central uses for input. Sampler Central does not use any input so it's good to set to 0. This is also not displaying channels available in the soundcard.



Sample Rate


If for some reason you don't want to use 44100 as a sample rate you can change that here.



Hardware Buffer Size


This is the number of samples that SC processes at every tick of the CPU. Lower settings means more work for the CPU, but less latency in the sound. 1024 is generally a good setting. If you have CPU problems you can try to set this to a higher value.



Memory Size


At the moment there is no reason to change this since no effects or processes uses much of this memory...
